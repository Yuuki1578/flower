# Flower - Low-Level Static Site Generator

## Quick Start

1. ### Clone this repository
   ```bash
   git clone https://github.com/Yuuki1578/flower.git
   cd flower
   ```

2. ### Building
   Untuk build project, dibutuhkan _cmake_ dan _make_
   ```bash
   cmake --build .
   ```
